# Documentation du Projet Lebonprix

Ce dossier contient toute la documentation du projet, organisée de manière structurée.

- `AUTO_SAVE_TESTING_GUIDE.md` - Guide de test pour la sauvegarde automatique
- `CINETPAY_TESTING_GUIDE.md` - Guide de test pour l'intégration CinetPay
- `PWA_TESTING_GUIDE.md` - Guide de test pour PWA
- `PERFORMANCE_OPTIMIZATION_GUIDE.md` - Guide d'optimisation des performances
- `PWA_ICON_GUIDE.md` - Guide pour les icônes PWA
- `PWA_TROUBLESHOOTING.md` - Guide de dépannage PWA
- `EMAILJS_SETUP.md` - Guide de configuration EmailJS

- `IMPLEMENTATION_REPORT.md` - Rapport d'implémentation

- `CONFIGURATION_COULEURS_THEME.md` - Configuration des couleurs du thème
- `CORRECTIONS_CHARTE_GRAPHIQUE.md` - Corrections de la charte graphique
- `STOCK_MANAGEMENT_PROPOSAL.md` - Proposition de gestion des stocks

### `/docs/migrations/`

- Rapports de migration de company IDs
